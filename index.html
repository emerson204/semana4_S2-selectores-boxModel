Selectores

Selector de etiqueta

El selector de etiqueta es el de menor especificidad , sirve para dar estilos generales como al inicio del proyecto

Selector de clase

El selector de clase es el que mas utilizaremos en dar estilos a las etiquetas , es como un identificador a la etiqueta que se le va a dar estilo , se llama con un .nombreClase

Selector id

El selector Id es un identificador unico en las etiquetas osea si tu le pones un nombre de id a un etiqueta , ese nombre no debe de ser llamado en otra etiqueta , se llama con #nombreId
